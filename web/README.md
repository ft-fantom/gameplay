# fccweb

## Quick start

1. Clone the project

```bash
# Clone the project
$ git clone https://github.com/findtruman-co-creation/fccweb.git
$ cd fccweb
```


2. Install dependencies

```bash
# Install dependencies
$ pnpm install
```


3. Start the dev server

```bash
# Start the dev server
$ pnpm dev
```

4. Build the project

```bash
# Build the project
$ pnpm build
```
